Object Oriented Programming

Java is known as an object oriented programming language.

This means that it is easy to represent entities as objects by using class and encapsulation

An example of this might be a Student class to represent a student 

                
                    public class Student{
                        /* Student properties */
                        private String name
                        private int age;

                        /*Constructor */
                        public Student (String name, int age){
                            this.name = name
                            this.age = age;
                            
                        }
                        
                        /*Getter method */
                        public String getName() {
                            return name;
                        }

                        /* Setter method */
                        public void setName(String name) {
                            this.name = name
                        }

                    }
